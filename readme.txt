== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `mime-types-extended` and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enable mimetypes in the options dialog.
